# Legal Notes

## What this project is

Daffy Editor is an original, clean-room tool for inspecting and (eventually)
editing level data of the SNES game *Daffy Duck: The Marvin Missions*. It is
written from scratch in Rust and contains **no code or assets from the game**.

## What this repository must never contain

- ROM images, in whole or in part (including "test" ROMs of the real game)
- Extracted graphics, music, text, level data, or any other game assets
- Modified ROMs
- Hash-reversible encodings of any of the above

Automated tests use only synthetic data generated by the test code itself.

## User obligations

- You must own a legally obtained copy of the game and produce your ROM image
  yourself by means that are lawful in your jurisdiction.
- The editor reads the ROM from a file *you* select; it never downloads one.

## Distribution of work

- Share your hacks as **IPS or BPS patches** (File > Export IPS/BPS Patch).
  Patches contain only your changed bytes plus offsets/checksums — not the game.
- "Export Modified ROM" exists strictly as a local convenience for testing in
  your own emulator. Do not distribute its output.

## Identification data

The project stores and publishes only *identifying metadata* about the ROM
(sizes, CRC32/SHA-1 hashes, offsets, structure documentation). Such facts are
not copyrightable content and are standard practice in the preservation and
ROM-hacking communities (cf. No-Intro database).
